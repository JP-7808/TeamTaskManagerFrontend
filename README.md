# 🚀 Team Task Manager Frontend

A modern and responsive **Task Management Web Application Frontend** built using React.
This application helps teams efficiently manage tasks, assign responsibilities, and track progress in real-time.

---

## 📌 About the Project

The **Team Task Manager Frontend** provides an intuitive UI for managing tasks within a team environment. It connects with a backend API to deliver full-stack functionality.

### 🔹 Key Capabilities:

* Create and manage tasks
* Assign tasks to team members
* Track progress and status
* Visualize task data in dashboards

---

## 🛠️ Tech Stack

### 💻 Frontend Technologies:

* ⚛️ React.js
* 📜 JavaScript (ES6+)
* 🎨 Tailwind CSS
* 🔗 Axios (API calls)
* 🧭 React Router DOM

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 📋 Create, Update & Delete Tasks
* 👥 Assign Tasks to Team Members

* 🔎 Search & Filter Tasks
* 📱 Fully Responsive Design

## ⚙️ Getting Started

### 📋 Prerequisites

Make sure you have installed:

* Node.js (v14 or above)
* npm or yarn
* Git

---

### 🔧 Installation

Clone the repository:

```bash
git clone https://github.com/JP-7808/TeamTaskManagerFrontend.git
```

Navigate to the project folder:

```bash
cd TeamTaskManagerFrontend
```

Install dependencies:

```bash
npm install
```

---

### ▶️ Run the Application

```bash
npm run dev
```

The app will run at:

```
http://localhost:3000
```
